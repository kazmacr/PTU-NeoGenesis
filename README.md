# Pokémon Tabletop United: Neo Genesis (Sistema para Foundry VTT)

Este repositorio contiene el código fuente para el sistema de juego no oficial de **Pokémon Tabletop United: Neo Genesis (Edición en Español)** para la plataforma [Foundry Virtual Tabletop](https://foundryvtt.com/).

## Descripción del Proyecto
Este sistema está diseñado desde cero para automatizar y agilizar las mecánicas específicas de la versión Neo Genesis, manejando la alta densidad de datos que requiere el juego. 

**Características principales (En desarrollo):**
* Diferenciación estricta de Estadísticas (Base, Añadidas, Etapas de Combate y Naturalezas).
* Cálculo automático de Evasiones y Puntos de Salud máximos.
* Soporte nativo para Pokérasgos, Talentos y Rasgos de Clase.
* Automatización de tiradas basadas en las tablas de Base de Daño (DB) del manual Core.

## Estado del Proyecto
El sistema se encuentra actualmente en fase de **desarrollo inicial (Arquitectura de Datos)** y aún no es funcional para jugar partidas completas.
